The "llm-generated-data" folder contains all the data generated by the LLM.


files ending in "ABAC-rules.txt" are the abac rules generated by the LLM
files ending in "-ACL.txt" is the ACL list generated using the abac rules file.
files ending in ".log" are a log of the entire request made to the LLM, the LLM ABAC respsonse, 
    the LLM ACL, and the analytics performed on the rules.


each LLM will have its own folder with the above files
